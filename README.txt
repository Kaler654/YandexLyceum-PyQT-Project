Проект "Кинотеатры".
Суть проекта - имитация работы кинотеатров, как со стороны посетителя, так и со стороны разработчика с использование PyQT.
Основаная часть проекта состоит из 2 файлов(CinemaClasses.py и CinemaGUI.py), первый файл является каркасом проекта, в нём прописана большая часть функционала.
Второй файл отвечает за дизайн проекта и некоторую часть функционала, остальные файлы .py являются дизайном проекта.

Файл для запуска приложения - CinemaGUI.py
